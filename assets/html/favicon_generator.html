Instructions pour générer les favicons

1. Utilise un outil comme favicon.io ou realfavicongenerator.net

2. Prends une capture d'écran de la version 128x128 ci-dessus

3. Upload l'image et génère tous les formats nécessaires

4. Place les fichiers dans public/ de ton projet Laravel

5. Met à jour le <head> avec les liens vers les favicons

Code à ajouter dans le <head>

<!-- Favicons -->
<link rel="icon" type="image/x-icon" href="/favicon.ico">
<link rel="icon" type="image/png" sizes="32x32" href="/favicon-32x32.png">
<link rel="icon" type="image/png" sizes="16x16" href="/favicon-16x16.png">
<link rel="apple-touch-icon" sizes="180x180" href="/apple-touch-icon.png">
<link rel="manifest" href="/site.webmanifest">
